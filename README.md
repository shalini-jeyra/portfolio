# Ashlin Shalini - Senior Flutter Developer Portfolio

The source code for my personal professional portfolio, engineered natively with Flutter Web. 

View the live portfolio here: [https://shalini-jeyra.github.io/portfolio/](https://shalini-jeyra.github.io/portfolio/)

## 🚀 Tech Stack

- **Framework**: Flutter (Web rendering via CanvasKit)
- **Language**: Dart
- **Design System**: Custom Obsidian & Gold theme palette
- **Animations**: `flutter_animate`, `animated_text_kit`
- **Layout Engine**: Custom intrinsic wrapping and flexible layouts (bypassing strict GridView aspect ratios for robust data scaling).

## 📱 Key Features

- **Bulletproof Responsiveness**: Fluidly adapts across Desktop, Tablet, and Mobile viewports without relying on rigid CSS-style media breakpoints.
- **Dynamic Content Scaling**: Cards and components naturally map to the intrinsic height of their text, eliminating RenderFlex collisions.
- **Premium Aesthetics**: High-end deeply saturated gradients, scroll-reveal animations, and unified typography.

## 🛠 Building Locally

If you'd like to explore the source code locally:

```bash
# Clone the repository
git clone https://github.com/shalini-jeyra/portfolio.git

# Navigate into the directory
cd portfolio

# Install dependencies
flutter pub get

# Run on web (Chrome)
flutter run -d chrome
```
